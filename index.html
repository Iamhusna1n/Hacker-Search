README

Welcome to Hacker Search Terminal!

Search Features:

  • Type your query and press Enter to search
  • Use search engine tags to switch between engines
  • Quick search: Type engine name + space + query (e.g., "yt cats")

Themes:

  • Click the gear icon to open settings
  • Select from 5 themes: Matrix, Cyberpunk, Retro, Hacker, Neon
  • Theme changes affect all UI elements and matrix background

Clock Widget:

  • Toggle between digital and analog display
  • Change timezone in settings
  • Drag widget to reposition

Todo List:

  • Add tasks using the input field
  • Click cross to remove tasks
  • Tasks are saved automatically

Weather Widget:

  • Set your location in settings
  • Shows temperature, wind speed, and conditions

Widget Controls:

  • Drag widgets by their headers
  • Click _ to minimize
  • Click ↔ to resize
  • Minimized widgets appear on the left side

Media Controls:

  • Control audio playback
  • Play, pause, and stop functions

Matrix Background:

  • Changes color with theme selection
  • Only visible when online
  • Automatically adjusts to window size
